# Django Blog Management System

A feature-rich Blog Management System built with Django, designed for learning and real-world blogging applications. This project includes role-based access control, blog management, category management, commenting, media uploads, dashboards, and search functionality.

## 🚀 Features

### User Management
- User Registration & Login
- Authentication & Authorization
- Role-Based Access Control
- Django Groups & Permissions

### Blog Management
- Create, Read, Update, and Delete Blogs
- Blog Categories
- Featured Posts
- Recent Posts Section
- Slug-Based URLs
- Rich Blog Content

### Comment System
- Authenticated Users Can Comment
- Comment Management
- Blog Discussion Support

### Dashboard
- Admin Dashboard
- Manager Dashboard
- Editor Dashboard
- Blog Statistics
- User Statistics
- Category Statistics

### Search & Navigation
- Blog Search Functionality
- Pagination
- Category Filtering

### Media Handling
- Blog Image Uploads
- Media File Management
- Static File Configuration

## ⚙️ Installation

```bash
git clone https://github.com/AjinCodeHub/Django_Blog.git
cd Django_Blog
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
